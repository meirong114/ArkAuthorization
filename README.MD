# ArkAuthLegacy 

---

### 这是什么？
> 一个基于安卓的《明日方舟》通行认证挂件的实现

> 最低支持Android 4.4，你应该需要拆了废旧手机来做

### 我怎么用？
> 将你的成品通行证视频(1080*1920,不然会有拉伸)扔到手机存储的DCIM文件夹内的Authorization文件夹，然后设置默认桌面，授予权限，ok！

### 想改程式？
> 把这个仓库Fork到你那，自己慢慢改，等改到差不多了你就可以提交一下PR看我心情好不好(划掉)

### 有野要讲？
> 你可以去到Issues里面慢慢讲
